# 🤖 A Origem do Mascote Android

Um site informativo e responsivo que conta a história e as curiosidades por trás da criação do mascote do sistema operacional Android.

Projeto desenvolvido integralmente durante o módulo de HTML5 e CSS3 do Curso em Vídeo (Prof. Gustavo Guanabara), aplicando na prática os conceitos de semântica web e estilização.

**Tecnologias utilizadas:**
* HTML5
* CSS3
